Chart Object

The createChart function creates a chart view object. The object has functions and properties that allow you to control the plots that are rendered in the chart.

    var chart = charts.createChart({ ... });

Properties

  • plots[array]: Array of plots currently assigned to the chart object

Functions

createLinePlot

This function creates a new line plot object and adds it to the chart. See Line Plot for parameters of this function.

createBarPlot

This function creates a new bar plot object and adds it to the chart. See Bar Plot for parameters of this function.

createPiePlot

This function creates a new pie plot object and adds it to the chart. See Pie Plot for parameters of this function.

add

Adds one or more plots to the chart object.

Arguments

Takes one argument, either a plot object or an array of plot objects.

    chart.add( plot1 ); 

    - or -

    chart.add([ plot1, plot 2, plot3 ]);

remove

Removes a plot from the chart object.

Arguments

Takes one argument, a plot object to remove.

    chart.remove( plot1 );
